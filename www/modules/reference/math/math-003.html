WORKING WITH FRACTIONS


The top number of a fraction is called the numerator.

The bottom number of a fraction is called the denominator.

To multiply:

1⁄2 x 3⁄4 = 1 x 3⁄2 x 4 = 3⁄8

To divide, multiply the first fraction with the reciprocal of the other:

1⁄2 ÷ 3⁄5 = 1⁄2 x 5⁄3 = 1 x 5⁄2 x 3 = 5⁄6

To add or subtract, first find the common denominator:

1⁄3 + 2⁄5 = (1 x 5⁄3 x 5) + (2 x 3⁄5 x 3) = 5⁄15 + 6⁄15 = 11⁄15
